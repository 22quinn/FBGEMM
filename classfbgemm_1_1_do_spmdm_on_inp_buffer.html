Detailed Description

template<typename outT = std::int32_t, typename inT = std::int32_t, typename nextOPType = DoNothing<inT, inT>>
class fbgemm::DoSpmdmOnInpBuffer< outT, inT, nextOPType >

SPMDM (SParse Matrix times Dense Matrix) inplace on the 32-bit input buffer (inp). After modifying the input buffer, pass it to the next op. When groups > 1, each group is numRows() x (numCols()/groups) matrix.
