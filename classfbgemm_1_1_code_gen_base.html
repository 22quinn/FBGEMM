Get or Create the instructions for macro-kernel.

If the problem size (mc, nc) and accumulation flag (accum) can be found in the code cache (a hash map), then get the macro-kernel instructions directly from it. Otherwise, create the instructions for macro-kernel, and store that into the code cache.
