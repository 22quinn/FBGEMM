Behavior is as the follow pseudocode (when use_offsets == true, lengths[i] == offsets[i + 1] - offsets[i]) (when is_weight_positional == true, use weights[j - offsets[i]] instead of weights[j])

for i in range(output_size): out[i * block_size : (i + 1) * block_size] = 0 for j in range(offsets[i], offsets[i + 1]): for k in range(block_size): out[i * block_size + k] += input[indices[j] * block_size + k] * weights ? weights[j] : 1; if normalize_weights and lengths[i] > 0: out[i * block_size : (i + 1) * block_size] /= lengths[i]
